# ScamShield AI 🛡️

An AI-powered web application that helps users identify potential online scams by analyzing suspicious messages and URLs, providing instant risk assessments and cybersecurity awareness.

Built as a **hackathon project** by a 3-member team at Kristu Jayanti (Deemed to be University).

## Features
- Analyze suspicious messages and URLs for scam indicators
- AI-powered risk assessment with instant results
- Simple, user-friendly web interface
- Deployed and live on Render

## Tech Stack
- **Backend:** Python, Flask
- **Frontend:** HTML, CSS, JavaScript
- **AI:** AI APIs for scam detection logic
- **Database:** SQL
- **Version Control:** Git & GitHub
- **Deployment:** Render

## Project Structure
```
scamshield-ai/
├── app1.py           # Main Flask application
├── static/           # CSS, JS, and static assets
├── templates/        # HTML templates
├── requirements.txt  # Python dependencies
└── .gitignore
```

## How to Run Locally
1. Clone the repository
   ```
   git clone https://github.com/Narendran-11/scamshield-ai.git
   ```
2. Install dependencies
   ```
   pip install -r requirements.txt
   ```
3. Run the app
   ```
   python app1.py
   ```
